# WFX AI-Native ERP

An AI-powered ERP dashboard for the apparel sourcing industry built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**. The application demonstrates how AI can simplify ERP workflows through natural language querying, intelligent product discovery, semantic image search, and interactive business analytics.

Ask business questions in plain English.

Examples:

* Show cotton shirts supplied by ABC Textiles
* Find black hoodies under ₹1500
* Show pending invoices
* Top buyers by revenue
* Which supplier has the highest rating?
* Buyers purchasing above 220 GSM

The application generates:

## Getting Started

### Clone the Repository

```bash
git clone <repository-url>
cd <project-name>
```

### Install Dependencies

```bash
pnpm install
```

### Start Development Server

```bash
pnpm dev
```

Open:

```
http://localhost:5173
```

## Available Scripts

